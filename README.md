# Portfólio Pessoal — Maria Izabel de Lima

## Sobre o projeto

Esse projeto é o meu portfólio pessoal. Criei ele para me apresentar, mostrar minhas habilidades e os projetos que já desenvolvi. 

## Sobre mim

Meu nome é Maria Izabel de Lima, moro em Balneário Camboriú, SC. Estou no início da minha jornada como desenvolvedora web e estudo programação todos os dias. Tenho muito interesse em continuar crescendo na área de tecnologia.

## O que tem no site

O site tem quatro partes principais:

A primeira é a apresentação, onde coloco meu nome e falo um pouco sobre quem eu sou e o que estou estudando.

A segunda é a parte de habilidades, onde mostro minhas habilidades comportamentais como proatividade, trabalho em equipe, aprendizado contínuo e atenção aos detalhes.

A terceira é a parte de projetos, onde listo os projetos que já desenvolvi com o nome de cada um, uma descrição e as tecnologias que usei.

A quarta é o contato, onde deixo meu e-mail e link do GitHub para quem quiser falar comigo.

## Tecnologias que usei

Para fazer esse portfólio eu usei HTML para criar a estrutura da página, CSS para deixar tudo com a aparência que eu queria, e JavaScript para gerar os cards de projeto automaticamente na tela.
